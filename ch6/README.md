# CH6 위치찾기

> 텍스트의 특정 위치를 찾는다.
>


### 단어 경계 지정하기


| 메타 단어 | 설명 |
| :---: | :---: |
| \b | 단어의 시작이나 마지막을 일치 |


> ***ex***:
	\bcat\b
>

> vim:
	/\<cat\>/
>

> \b 는 일반적으로 단어의 일부로 사용하는 문자( \w ) 와 그 외의 문자( \W ) 사이에 있는 위치와 일치한다.
>

> \b 와 \B 에 대해서 이해가 힘들게 설명되어 있다. 밑의 링크를 보자. 설명 잘해놓았다.   
	꿀벌개발일지: <https://ohgyun.com/392>
>


### 문자열 경계 정의하기


| 메타 단어 | 설명 |
| :---: | :---: |
| ^ | 문자열 경계 시작 |
| $ | 문자열 경계 끝 |



> ***ex***:   
	<\?xml.?>   
	^\s<\?xml.?>   
	</[Hh][Tt][Mm][Ll]>\s*$
>


### 다중행 모드( multiline )



| 메타 단어 | 설명 |
| :---: | :---: |
| (?m) | 다중행 모드로 변경하면 정규표현식 엔진이 줄바꿈 문자를 문자열 구분자로 인식한다.  |



> ***ex***:   
	(?m)^\s*//.*$  
	(?m)^\s*//.*$(?m)  
>

> (?m) 을 구현하지 않는 곳도 많다.   
	적어도 자바스크립트는 안되더라.
>
