# CH7 하위 표현식 사용하기

> Subexpression( 하위 표현식 ) 은 여러 표현식을 묶는다.
>


### 하위 표현식 이해하기

| 메타 문자 | 설명 |
| :---: | :---: |
| 표현식(표현식)표현식  | 큰 표현식 안에 속한 일부 표현식을 한 항목으로 묶음 | 
| 표현식\|표현식| OR 연산자 | 

>
***ex***:   
	(&nbsp;){2,}   
	(\d{1,3}\.){3}\d{1,3}
	(19|20)\d{2}
>

### 중첩된 하위 표현식

>
***ex***:   
	(((\d{1,2})|(1\d{2})|(2[0-4]\d)|(25[0-5]))\.){3}((\d{1,2})|(1\d{2})|(2[0-4]\d)|(25[0-5]))
>
